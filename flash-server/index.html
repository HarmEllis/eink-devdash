eink-devdash

Disconnected

1. Flash firmware (first time only)

Install or update the firmware over USB. Skip if the device already runs eink-devdash.

2. First-time provisioning

After flashing, the e-ink display shows a temporary WiFi network such as devdash-A4F2 and its password. Join that network, open http://192.168.4.1, then enter WiFi SSID, WiFi password, API URL, and device token. The device saves the config, restarts, joins WiFi, fetches the API, and renders the dashboard.

3. Connect Serial

Optional recovery/editor path: open a WebSerial link to the running firmware so this page can read and write config via Improv RPCs.

4. Load current config

Reads the saved WiFi networks and API endpoints back from the device. Tokens come back masked as ****xxxx — re-enter a token to overwrite, leave the masked value to keep the stored one.

Reprovision WiFi wipes the saved WiFi/API profile and reboots the device into its built-in SoftAP portal.

5. Network editor

Edit any field — the card you are editing is highlighted and used by Save below. Each save or delete triggers a device restart; Connect Serial again afterwards.

6. Save / apply
